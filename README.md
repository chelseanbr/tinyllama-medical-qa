# aih-final-project

## Idea(s)
- Multi-Modal ICU Patient Outcome Prediction
  - Task: Combine clinical notes, time-series vitals, and imaging data (if available) to predict outcomes like survival or complications.
  - ML Approach: Multimodal deep learning (BERT for text + CNNs for images + LSTMs for vitals).
  - Impact: Provides a comprehensive risk assessment.
