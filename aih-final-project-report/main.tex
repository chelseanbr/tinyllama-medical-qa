\section{Introduction}

Medical question answering is a challenging task due to the complexity of medical language and the need for accurate information.
In this project, I generate multiple-choice questions (MCQs) from the MedQuAD layperson question answering dataset using rejection sampling.
The objective is to create a set of MCQs that can effectively assess layperson medical knowledge.
Additionally, I perform supervised fine-tuning of the TinyLLaMA model using the curated MCQs to improve its performance in answering layperson medical questions.
