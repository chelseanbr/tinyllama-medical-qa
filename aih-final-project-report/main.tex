\section{Introduction}
Medical question answering is a challenging task due to the complexity of medical language and the need for accurate information.
In this project, I focus on generating multiple-choice questions (MCQs) from the MedQuAD layperson Question Answering dataset via Rejection Sampling.
The goal is to create a set of MCQs that can be used to assess layperson medical knowledge.
Additionally, I fine-tune the TinyLLaMA model to improve its performance on layperson questions.
